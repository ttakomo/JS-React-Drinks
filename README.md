# JS-React-Drinks
# JavaScript Programming example using json-server as backend and front-end implemented with react &amp; bootstrap
Instructions:

1.Start the json server:
  -  cd json-server 
  -  npm install 
  -  npm start
  
2.Start the front-end in another cmd prompt: 
  -  cd front
  -  cd drinks
  -  npm install
  -  npm start
